SNMP Scanner
============

##Requirments:
	1. scapy
	```bash
	sudo apt-get install python-scapy
	```
	
	2. libevent (requirement for gevent)
	```bash
	sudo apt-get install libevent-dev
	```
	
	3. gevent
	```bash
	sudo apt-get install gevent
	```

##Usage:
Needs root to run!
```
sudo snmpscan.py
```

snmp scan a network!
